簡単なプロッタ

紹介

このプログラムは簡単な描画命令を標準出力から出力するだけで グラフィクスを描くプロッタプログラムです。

実行環境

Java 1.5 以上の Java の実行環境で Swing アプリケーションが動作 する環境（Java の実行環境をインストールした Windows や Linux など） で動作します。

実行

java （Java のVM を実行するコマンド）にパスが通っているものとし、 コマンドプロンプト(Windowsの場合)で、本ディレクトリに移動して
  • 標準出力から描画命令を出力するプログラムと連動させる： 
    実行コマンド | java SimplePlotter
  • 描画命令を書き出したファイルを読み込む： 
    java SimplePlotter < 描画命令ファイル
    • 終了するにはメニューバーの File メニューから exit を選びます。

描画空間

描画空間は 400 × 400 ピクセルに固定されており、座標系 (x,y) は左上が (0,0), 右下が (399,399) です。

描画命令

１つの描画命令は１行に書かなければなりません。小文字、大文字は区別されます。 以下の描画命令が実装されています。なお、 ここで (x1) など () をつけて書かれたものは実際には数値表現です。
white
描画する色を白にします。同様に色指定として black, red, green, blue, yellow が利用できます。
line (x1) (y1) (x2) (y2)
座標 (x1,y1) から座標 (x2,y2) まで線を引きます。
circle (x1) (y1)
座標 (x1,y1) に小さな円を描きます。同様な命令として fillCircle, square, fillSquare があります。
message [...]
描画面の下側に[...]で書かれた文字列を出力します。
flush
個々の描画命令は直ちに画面には反映されないことがあります。flush 命令でそれまでの描画命令を 画面に確実に反映させます。
clear
画面を消去します。
pause
画面に次に進んでよいかどうがを尋ねるダイアログを出し、ボタンが押されるまで待ちます。
